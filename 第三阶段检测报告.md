## 二、AI 智搜页面功能详解

### 2.1 页面布局

| 区域 | 功能 | 交互 |
|------|------|------|
| 搜索头部 | 深色渐变背景 + 搜索框 + 快捷标签 | 输入查询 / 点击标签快捷搜索 |
| 加载动画 | 三点弹跳动画 + 趣味文案 | 搜索中自动显示 |
| AI 推荐语 | 卡片式展示，带打字机光标 | 逐字渲染 AI 响应 |
| 技能卡片列表 | 左图右文卡片，含标签和价格 | 点击跳转详情页 |
| 空状态/错误 | 友好的空结果和错误提示 | 支持重试 |

### 2.2 SSE 流式处理核心逻辑

```
用户点击搜索
    │
    ▼
fetchSSE('/api/ai/search-stream', { query })
    │
    ▼ (SSE 流式返回)
onChunk(chunk) → recommendationText += chunk  ← 逐字追加，打字机效果
    │
    ▼ (流结束)
onDone() → parseResult()
    │
    ▼ (JSON 解析)
提取 recommendationText + skillIds
    │
    ▼
fetchSkillDetails(skillIds) → 渲染技能卡片列表
```

### 2.3 关键特性

| 特性 | 实现方式 |
|------|---------|
| 打字机效果 | SSE onChunk 逐字追加到 recommendationText，配合 CSS cursor 闪烁动画 |
| 加载动画 | CSS @keyframes bounce 三点弹跳 + 趣味文案 |
| JSON 容错 | 清理 markdown 代码块标记后解析，解析失败保留原始文本 |
| 快捷搜索 | 点击标签直接填充 query 并触发搜索 |
| H5/小程序兼容 | 条件编译：H5 用 fetch API 流式读取，小程序用同步请求 |

## 五、使用说明

### 5.1 项目初始化

由于本前端是纯 Vue 文件（非完整 UniApp 工程），使用方式有两种：

**方式 A：复制到 UniApp 项目**
1. 使用 HBuilderX 创建 UniApp + Vue3 项目
2. 将 `opc-frontend/` 下的文件复制到项目对应目录
3. 运行到浏览器或小程序

**方式 B：独立预览（H5 模式）**
1. 安装 Vue CLI：`npm install -g @vue/cli`
2. 创建 UniApp 项目：`npx degit dcloudio/uni-preset-vue#vite-ts opc-app`
3. 替换 src 目录内容为本前端代码
4. `npm install && npm run dev:h5`

### 5.2 后端跨域配置

前端运行在 localhost:5173，后端运行在 localhost:8080，需要配置跨域。

在 `sky-server` 的 `WebMvcConfiguration.java` 中添加：

```java
@Override
public void addCorsMappings(CorsRegistry registry) {
    registry.addMapping("/**")
            .allowedOriginPatterns("*")
            .allowedMethods("GET", "POST", "PUT", "DELETE", "OPTIONS")
            .allowedHeaders("*")
            .allowCredentials(true)
            .maxAge(3600);
}
```

### 5.3 测试流程

1. 启动后端 SkyApplication
2. 启动前端开发服务器
3. 打开 AI 智搜页面
4. 输入"我想找心理咨询师"或点击快捷标签
5. 观察：加载动画 → AI 推荐语逐字出现 → 技能卡片列表渲染
6. 点击技能卡片 → 跳转详情页
7. 详情页点击"立即购买" → 下单
