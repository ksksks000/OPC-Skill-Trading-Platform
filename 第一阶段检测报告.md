## 五、编译环境说明

当前终端环境未配置 JAVA_HOME，无法执行 `mvn compile`。请在本地 IDE 中执行以下验证步骤：

1. 确保 JDK 17 已安装并配置
2. 执行 `mvn clean compile` 验证编译
3. 在 MySQL 中执行 `sql/opc_skill_init.sql` 建表
4. 启动 SkyApplication，验证以下接口：
   - `POST /admin/employee/login` - 管理员登录
   - `POST /admin/skill` - 新增技能
   - `GET /admin/skill/page` - 分页查询技能
   - `GET /admin/skill/{id}` - 查询技能详情
   - `PUT /admin/skill` - 修改技能
   - `POST /admin/skill/status/{status}` - 启用/禁用技能
   - `DELETE /admin/skill?id=xxx` - 删除技能

## 六、已知遗留项

| 遗留项 | 优先级 | 说明 |
|--------|--------|------|
| HttpClientUtil 依赖 Apache HttpClient | 低 | 建议后续替换为 Spring RestTemplate/WebClient |
| PageResult 泛型化 | 低 | 当前使用原始 List，建议改为 PageResult\<T\> |
| BaseContext ThreadLocal | 中 | 微服务化时需替换为显式参数传递 |
| WebSocket 会话管理 | 低 | 使用 HashMap 非线程安全，建议改为 ConcurrentHashMap |
| application-dev.yml 硬编码密钥 | 中 | 建议使用环境变量或配置中心 |
| 支付模块 | 高 | 当前为模拟支付，需接入真实支付网关 |
